💸 Gider Takip Sistemi (Expense Tracking System)
Bu proje, kurum içi harcamaların dijital ortamda kayıt altına alınması, yönetilmesi ve raporlanması amacıyla geliştirilmiş bir Full-Stack uygulamadır. Kullanıcılar harcama talepleri oluşturabilir, fiş/fatura yükleyebilir; yöneticiler ise bu talepleri onaylayıp toplu işlemler gerçekleştirebilir.

✨ Öne Çıkan Özellikler
🔐 Güvenli Kimlik Doğrulama: Spring Security ve JWT (JSON Web Token) tabanlı oturum yönetimi.

👥 Rol Bazlı Yetkilendirme: * Admin: Kullanıcı yönetimi, toplu harcama onaylama (Bulk Transactions) ve detaylı raporlama.

Personel: Harcama talebi oluşturma, fiş/makbuz yükleme ve işlem geçmişi takibi.

📊 Finansal Raporlama: Harcamaların kategorilere ve tarihlere göre analiz edilmesi.

🖼️ Dosya Yönetimi: Yüklenen fişlerin/makbuzların sunucu tarafında güvenli saklanması.

🌓 Tema Desteği: Dark Mode uyumlu modern React arayüzü.

🛠️ Teknik Mimari
Backend (Java/Spring Boot)
Framework: Spring Boot

Güvenlik: Spring Security & JWT

Veri Erişimi: Spring Data JPA (Hibernate)

Veritabanı: PostgreSQL / MySQL

İletişim: REST API & DTO Pattern

Frontend (React.js)
Kütüphane: React

Durum Yönetimi: Context API

Stil: CSS3 (Custom Modules) & React Hooks

HTTP İstekleri: Axios
